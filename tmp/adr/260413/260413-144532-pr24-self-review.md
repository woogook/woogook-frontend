- reviewer 관점에서 diff, 관련 코드, 테스트, 정적 검증을 함께 확인했다.

# 변경 사항

- `src/features/local-council/detail.ts`에서 사용되지 않는 helper를 제거해 PR 범위 lint warning을 없앴다.
- `tests/local_council_proxy.test.ts`에서 streaming relay 회귀 테스트의 backend mock을 실제 `Response` 인스턴스로 바꿔 branch 자체 TypeScript 오류를 제거했다.

# 비채택안

- `npm run lint`에서 남아 있는 `src/app/components/CandidateCards.tsx`의 `<img>` warning은 현재 PR diff 바깥의 기존 이슈라 이번 self-review 조치 범위에는 포함하지 않았다.
- `npx tsc -p tsconfig.json --noEmit`는 PR 범위와 무관한 `assembly` 및 `observability` 쪽 기존 오류들 때문에 여전히 전체 저장소 기준으로는 실패한다.
